# Отчёт по практике LeetCode (Python)

В ходе практики были разобраны следующие темы и приёмы:

##  Темы:
1. **Списки и базовые циклы**  
   - Перебор элементов, `enumerate`, изменение списков.

2. **Списки и индексы**  
   - Работа с индексами, слияние упорядоченных списков.

3. **Множества (set)**  
   - Проверка дубликатов, операции `add`, `in`, `set()`.

4. **Пересечения и уникальные элементы**  
   - Пересечение множеств, отслеживание уникальных значений.

5. **Словари (dict)**  
   - Подсчёт частот, использование `dict.get`, `Counter`.

6. **Функции и генераторы**  
   - Создание функций, возврат значений, генераторы списков.

7. **Множества + словари**  
   - Хранение индексов, частотный анализ.

8. **Продвинутые словари**  
   - `defaultdict`, сортировка по значениям.

9. **Комбинация структур данных**  
   - Префиксные суммы, оптимизация с помощью словарей.

10. **Функции + множества + логика**  
    - Сортировка, множественные сравнения, чистые функции.
